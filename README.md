# 832301309_concacts_frontend
项目简介：本项目为通讯录管理系统（Contacts Management System）的前端部分，采用 HTML + Bootstrap +  JavaScript 实现页面设计与数据交互。  
系统支持 用户注册、登录、联系人增删编辑 功能，并通过 API 接口与后端（Flask）进行数据交互，实现前后端分离的完整 Web 应用。

2.vscode：运行app.py文件--打开index.html在default browser里面；
